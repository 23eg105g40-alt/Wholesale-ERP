# Wholesale ERP System

A full-stack MERN-based Wholesale ERP application designed to streamline inventory management, customer management, billing, and business reporting. The system provides role-based access control, invoice generation, PDF export functionality, and real-time inventory updates through an intuitive modern dashboard.
